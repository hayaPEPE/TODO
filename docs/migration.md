# 他PCへの移行手順書

## 前提

- 移行元PCと移行先PCの両方で Edge / Chrome などのモダンブラウザが利用できること。
- Node.jsや追加の外部パッケージは不要。
- TODOとラベルはブラウザの `localStorage` に保存されるため、アプリフォルダのコピーだけではTODOデータは移行されない。

## 移行するもの

- アプリ一式: この `todo` フォルダ全体
- TODOデータ: 画面の `JSON出力` で作成した `.json` ファイル

## 移行元PCでの作業

1. `index.html` をブラウザで開く。
2. 左サイドバー下部の `JSON出力` を押す。
3. `todo-export-YYYY-MM-DD.json` が保存されたことを確認する。
4. `todo` フォルダ全体と、出力したJSONファイルをUSBメモリや社内共有フォルダなどで移行先PCへコピーする。

## 移行先PCでの作業

1. 任意の作業フォルダに `todo` フォルダを配置する。
2. `todo` フォルダ直下の `index.html` をブラウザで開く。
3. 左サイドバー下部の `JSON取込` を押す。
4. 移行元PCで出力した `todo-export-YYYY-MM-DD.json` を選択する。
5. 上書き確認が表示されたら、内容を確認して承認する。
6. TODOとラベルが移行されていることを確認する。

## 動作確認

- TODO一覧が移行元PCと同じ件数になっている。
- ラベル欄に追加済みラベルが表示されている。
- 新規追加、編集、完了チェック、JSON出力が動作する。

## 注意事項

- CSV出力は確認・集計用であり、TODOデータの復元には使用しない。
- 移行先PCで既にTODOを入力している場合、JSON取込を実行すると現在のTODOとラベルは上書きされる。
- ブラウザを変えると `localStorage` も別管理になるため、通常利用するブラウザでJSON取込を実行する。
