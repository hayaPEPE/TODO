# TODOアプリ

ブラウザだけで動く、業務PC向けのTODOアプリです。Node.jsや外部パッケージのインストールは不要です。

## 起動方法

`index.html` をブラウザで開きます。

アプリ本体は `public/index.html` です。フォルダ直下の `index.html` を開くと自動でアプリ本体へ移動します。

## ファイル

- `index.html`: ブラウザ直接起動用の入口
- `public/`: 画面とフロントエンド実装
- `docs/design.md`: 設計書
- `docs/migration.md`: 他PCへの移行手順書

## データ保存

TODOとラベルはブラウザの `localStorage` に保存されます。同じブラウザで開くと前回の状態が復元されます。

TODOの期限は未指定でも保存できます。

TODOのラベルも指定なしで保存できます。追加したラベルは削除でき、削除したラベルが設定されていたTODOは指定なしに変わります。

表示件数、検索、フィルター、ラベル選択、期限ソートも同じブラウザ内で保持されます。

## データ出力

サイドバー下部からJSON形式とCSV形式でTODOを出力できます。別PCへ移行する場合は、JSON出力とJSON取込を使用します。

## 他PCへの移行

手順は `docs/migration.md` を参照してください。
